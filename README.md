# Market Sentiment Analysis: Fear vs Greed

📊 Project Overview

Financial markets are strongly influenced by investor sentiment, which often fluctuates between periods of Fear and Greed. These emotional market states can affect trading behavior, profitability, and market volatility.
This project performs Exploratory Data Analysis (EDA) to examine how trading performance changes under different market sentiment conditions. By comparing trading outcomes during Fear and Greed market phases, the analysis highlights patterns that help explain how sentiment impacts trading results.

🎯 Objectives

Analyze trading performance during Fear vs Greed market conditions

Compare average profitability (PnL) across sentiment phases

Examine profit variability and risk

Explore daily trends in trading performance
